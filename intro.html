What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create and structure content on the web. HTML uses tags to define elements such as headings, paragraphs, images, links, and more.

Example:

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>
  </body>
</html>

History of HTML

HTML was invented by Tim Berners-Lee in 1991. It has evolved through several versions:

  • HTML 1.0 – The first and most basic version.
  • HTML 4.01 – Released in 1999, with many structural tags.
  • HTML5 – The current version, supports multimedia and semantic elements.

HTML5 introduced new features like:

  • Video and audio support
  • New input types (e.g. date, color, range)
  • Semantic tags like <section>, <article>, <nav>

How HTML Works with Browsers

Web browsers (like Chrome, Firefox, Safari) read and interpret HTML files to display web pages. When you open an HTML file, the browser uses the tags to understand how the page should look and function.

The browser turns the HTML code into a visual layout that users can interact with.
